# 📈 株アシスタント (kabu-assist)

株式投資初心者のための「知識の補佐」ツール。GitHub Actions上で無料稼働し、毎朝のブリーフィングと市場急変時のアラートをDiscordに届けます。

## できること

- **☀️ デイリーブリーフィング (毎朝7:45)**: 日米の市況・ウォッチリスト銘柄の動き・直近イベントの注意点をまとめて投稿
- **🔗 サプライチェーン連想**: 「AIが伸びる → GPUのNVIDIAだけでなくHBMのマイクロン、装置の東京エレクトロン、電線のフジクラも」といった、思いつきにくい周辺銘柄を毎日1テーマ深掘り
- **🚨 急変動アラート (市場時間中2時間おきにチェック)**: 指数±2%、ウォッチリスト銘柄±5%、VIX25超え、ドル円±1%などを検知したら、その影響が波及しそうなセクターの解説付きで随時通知
- **📅 イベントカレンダー**: FOMC・日銀会合・選挙などが近づくと事前に警告

## セットアップ手順

### 1. リポジトリを作成
このフォルダの中身をそのままGitHubの新しいリポジトリにpushします。
**パブリックリポジトリにするとActionsの実行時間が無制限で完全無料**です(プライベートだと月2,000分まで無料。この構成なら月300分程度なのでプライベートでも収まります)。

```bash
cd kabu-assist
git init
git add .
git commit -m "初期セットアップ"
git remote add origin https://github.com/あなたのユーザー名/kabu-assist.git
git push -u origin main
```

### 2. Discord Webhookを作成
1. 通知を受け取りたいDiscordサーバーのチャンネルで「チャンネルの編集 → 連携サービス → ウェブフック → 新しいウェブフック」
2. 「ウェブフックURLをコピー」

### 3. Anthropic APIキーを取得
1. https://console.anthropic.com/ でアカウント作成
2. クレジットをチャージ(最低$5。デフォルト設定のHaikuモデルなら月数十円程度の消費)
3. API Keysからキーを発行

### 4. GitHubにシークレットを登録
リポジトリの `Settings → Secrets and variables → Actions → New repository secret` で以下2つを登録:

| Name | Value |
|---|---|
| `ANTHROPIC_API_KEY` | 手順3のAPIキー |
| `DISCORD_WEBHOOK_URL` | 手順2のWebhook URL |

### 5. 動作確認
リポジトリの `Actions` タブ → 「デイリーブリーフィング」→ `Run workflow` で手動実行。Discordに投稿されれば成功です。

## カスタマイズ

- **監視銘柄の変更**: `config.json` の `watchlist` を編集(日本株は `7203.T` のように `.T` を付ける)
- **アラート感度**: `config.json` の `alert_thresholds` を編集
- **業界知識の追加**: `knowledge/sector_map.json` に新しいテーマ・銘柄・連動ロジックを自由に追記。**ここを育てるほどツールが賢くなります**
- **イベントの追加**: `knowledge/event_calendar.json` に決算日・選挙・金融政策会合などを追記
- **分析の質を上げる**: `config.json` の `claude.model` を `claude-sonnet-5` に変更(コストは数倍)
